GERTUKO - Local Commerce Web Platform

🛒 What is it?

GERTUKO is a full-stack web application designed to connect local artisans and farmers directly with consumers. It aims to promote sustainable, zero-kilometer commerce by providing local producers with an accessible digital storefront.

🛠️ Technologies Used

Backend: Java, Spring Boot (MVC, Spring Security, Spring Data JPA)

Frontend: HTML5, CSS3, Bootstrap 5, Thymeleaf

Database: MySQL

Real-Time & Analytics: WebSockets (STOMP), Python (NumPy/SciPy for stats)

✨ Key Features

Role-Based Access: Distinct interfaces and permissions for Buyers, Sellers, and Administrators.

Real-Time Chat: A WebSocket-based messaging system allowing direct communication and negotiation between users.

Sales Prediction Algorithm: Integration of a Python-based statistical module (Linear Regression) that estimates future sales based on search ranking and visibility.

Secure Architecture: Implements BCrypt password hashing, HTTPS self-signed certificates, and a Master-Slave database replication strategy.
